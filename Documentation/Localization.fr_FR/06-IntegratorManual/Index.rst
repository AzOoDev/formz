Manuel intégrateur
==================

.. only:: html

    Intégrer des formulaires utilisant Formz s'apparente énormément à une intégration Extbase/Fluid classique. Vous serez amené à utiliser tous les outils habituels : layouts, templates, partials, mais aussi les ViewHelpers de Formz.

    Vous devrez tout de même respecter certaines contraintes pour un fonctionnement optimal de Formz.
