Integrator manual
=================

.. only:: html

    Integrating forms using Formz is much like a classic Extbase/Fluid integration. You will have to use regular tools: layouts, templates, partials, but also Formz ViewHelpers.

    You will still have to respect some constraints for Formz to work correctly.
